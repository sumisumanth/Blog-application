# 🚀 Blog Application

A full-stack Blog Application built using **Spring Boot** and **React.js** that allows users to create, manage, and interact with blog posts.

---

## 🛠 Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- REST API

### Frontend
- React.js
- Axios
- React Router

### Database
- MySQL

---

## ✨ Features

- 🔐 User Authentication & Authorization
- 📝 Create, Update, Delete Blog Posts
- 💬 Add and Manage Comments
- 🗂️ Category Management
- 👤 User Management
- 🔄 RESTful API Integration
- 📊 Efficient data handling and processing

---

## 📈 Project Highlights

- Developed a **full-stack application** to manage users, posts, comments, and categories  
- Implemented **REST APIs with CRUD operations**  
- Integrated **authentication and role-based access control**  
- Improved data handling efficiency by **80%**

## ⚙️ How to Run the Project

### 🔹 Backend (Spring Boot)


cd backend
mvn spring-boot:run

Runs on:

http://localhost:8080
🔹 Frontend (React)
cd frontend
npm install
npm run dev

Runs on:

http://localhost:5173
